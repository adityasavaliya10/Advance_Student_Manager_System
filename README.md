# 🎓 Advanced Student Tracker

A powerful command-line Student Management System built with Python and SQLite, designed to streamline student record management, academic performance analysis, attendance monitoring, and data reporting.

---

## 📖 Overview

Advanced Student Tracker is a comprehensive student management application that helps administrators efficiently organize and analyze student information. The project combines database management, academic analytics, ranking systems, scholarship evaluation, and data export capabilities into a single easy-to-use application.

The system demonstrates practical usage of Python programming concepts including:

* Object-Oriented Programming (OOP)
* Database Management with SQLite
* File Handling
* Data Analysis
* Error Handling
* CRUD Operations
* JSON & CSV Processing

---

## ✨ Features

### 👨‍🎓 Student Management

* Add new student records
* View all students
* Search students by ID or name
* Update existing student information
* Delete student records

### 📊 Academic Analytics

* Automatic percentage calculation
* GPA calculation system
* Performance evaluation
* Student ranking generation
* Topper identification

### 📅 Attendance Monitoring

* Attendance tracking
* Attendance-based scholarship analysis
* Student eligibility verification

### 🏆 Scholarship System

Students are automatically evaluated for scholarship eligibility based on:

* Academic Performance
* Attendance Percentage

### 📈 Dashboard Analytics

View key statistics including:

* Total students
* Average percentage
* Highest percentage
* Lowest percentage
* Number of failed students

### 💾 Data Management

* Export records to CSV
* Create JSON backups
* SQLite database storage
* Persistent student records

### 🔒 Authentication

* Admin login system
* Restricted access to management features

